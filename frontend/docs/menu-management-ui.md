# Menu Management UI Palette

This document captures the warm menu-management color system used across the menu screen.

## Colors

- Surface gradient: `#fff7ed` to `#fffdf7`
- Accent gradient: `#fb923c` to `#ef4444`
- Panel border: `#f4c7a7` and `#f1c9a8`
- Title text: `#492819`
- Body text: `#7c5b45`
- Price accent: `#8f2e11`

## Layout Notes

- Mobile-first layout.
- Two cards per row from medium screens upward.
- Four cards per row on large screens.
- Description stays collapsed by default to keep cards compact.

## Tailwind Usage

Use Tailwind utilities for layout, spacing, and responsive behavior where possible.

## Reference

This palette is the canonical reference for future menu-management UI work.
